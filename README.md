#项目名称：SRE Helper - 只能巡检工具

一个基于python的服务器健康巡检命令行工具，通过配合文件驱动，自动采集系统关键指标并声称结构化报告

##功能特性

- **系统资源检查**：cpu使用率、内存使用率、磁盘使用率
- **服务状态检查**：关键进程存活、端口监听状态
- **YAML 配置驱动**：检查项、阈值全部通过配置文件管理，无需改动代码
- **巡检报告输出**：终端可视化报告+日志文件持久化，方便追溯
- **可扩展架构**：新增检查类型只需添加一个函数和一个配置项

##技术栈

- Python 3.9+
- PyYAML
- subprocess

##快速开始

pip install pyyaml
python3 check.py

##项目结构
check.py     #主程序
config.yaml  #配置文件，定义检查项和阈值
reports/     #巡检报告存储目录

##设计思路
- YAML配置文件驱动，逻辑与数据分离
- 模块化检查函数，新增检查类型只需加函数和配置项
- 报告终端输出，日志文件持久化
